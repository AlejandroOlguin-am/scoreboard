# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Añadido
- Sistema base de detección y tracking de objetos
- Interfaz gráfica para control de partidas
- Comunicación serial con PIC18F4550
- Documentación inicial
- Tests unitarios básicos

### Cambiado
- Migración a Python 3.8+
- Actualización a OpenCV 4.x

### Arreglado
- Corrección en el cálculo de centroides
- Mejora en la estabilidad del tracking
- Optimización del uso de memoria

## [0.2.0] - 2023-12-20

### Añadido
- Implementación inicial del detector de bolas
- Clase base para tracking de objetos
- Protocolo de comunicación serial
- Configuración básica del proyecto

### Cambiado
- Reestructuración del código en módulos
- Mejora en la organización del proyecto

### Arreglado
- Problemas de detección en condiciones de baja luz
- Errores en la comunicación serial

## [0.1.0] - 2023-11-30

### Añadido
- Estructura inicial del proyecto
- Configuración básica de OpenCV
- Prueba de concepto del sistema de puntuación
